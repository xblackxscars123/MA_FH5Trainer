# xpaint

A lightweight color utility application for Windows.

## Features

- **Color Palette Management** — Create, save, and organize custom color palettes
- **Modern UI** — Clean, dark-themed interface built with WPF
- **Configurable** — Hotkey support and customizable settings

## Requirements

- Windows 10 or later (x64)
- .NET 8.0 Runtime
